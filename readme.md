# 🛍️ SmartCart

A modern full-stack e-commerce platform built using the MERN stack, providing a seamless online shopping experience with secure authentication, OTP-based email verification, product management, wishlist, shopping cart, and order processing.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/kurapati-mahendra/smartcart.git
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install --legacy-peer-deps
npm start
```

---

## 🔑 Environment Variables

Create a `.env` file inside the backend directory.

```env
PORT=8000

MONGO_URI=YOUR_MONGODB_URI

JWT_SECRET=YOUR_SECRET_KEY

EMAIL=YOUR_EMAIL

EMAIL_PASSWORD=YOUR_APP_PASSWORD

ORIGIN=http://localhost:3000
```
